目前只支持中国服务器，要使用这个挂载器需要在你第一次启动时设置你的游戏目录程序，然后把你的.lolskin文件拖放到程序界面中，双击你的皮肤卡片，皮肤卡片右下角显示红色，已挂载，标签表示wad文件已从.lolskin文件中导出到挂载目录，在点击窗口顶部，第9个按钮启动挂载功能即可，第8个按钮目前没有什么用，是用于预留外部自定义程序注入而备用，第7个按钮可用于直接打开你的wad文件挂载目录，可将wad文件直接放入到这个文件夹中，只要挂载功能是开启状态他将自动加载你的wad文件
